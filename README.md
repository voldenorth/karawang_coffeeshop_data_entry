Coffee Shop Data Entry – Karawang Regency 
Project Overview
This repository contains a manual data entry project of 50 coffee shops located in Karawang Regency, Indonesia.
All data was collected directly from Google Maps to ensure real-world accuracy and up-to-date information.
Each record includes essential business details and an address column with a clickable Google Maps hyperlink pointing to the exact shop location.

Dataset Information
Total records: 50 coffee shops
Columns included:
name – Coffee shop name
address – District-level address with Google Maps hyperlink
contact – Phone number (if available)
rating – Google Maps user rating
price_range_idr – Estimated price range per person (IDR)

Ratings are taken directly from Google Maps at the time of collection
Price ranges reflect commonly listed menu prices per person
No automated scraping tools were used; all entries were manually verified
